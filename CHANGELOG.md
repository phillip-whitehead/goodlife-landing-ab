# Goodlife Landing Page A/B Testing — Version Changelog

## Week 1 (March 22, 2026)

### V1 — Baseline
- **File:** `versions/v1.html`
- **Hypothesis:** Control group — current design with real photos
- **Changes:** 7-section layout (hero, trust bar, 6 properties, 6 islands, why us, testimonials, CTA+footer), real property and island photos from thegoodlifebahamasrentals.com, standard CTAs
- **Status:** Live

### V2 — Above-the-Fold Optimization
- **File:** `versions/v2-above-fold.html`
- **Hypothesis:** If we optimize the above-the-fold content with benefit-driven headline copy, a single prominent CTA, and credibility badges near the action button, conversion will improve because hero tests produce 50%+ swings (ThrillX data)
- **Changes:** Rewrote headline, consolidated to single CTA, added credibility badges next to action button, full-bleed hero image
- **Status:** Live

### V3 — Anxiety Reduction
- **File:** `versions/v3-anxiety-reduction.html`
- **Hypothesis:** If we add pricing transparency, cancellation policy badges, and trust signals near CTAs, inquiries will increase because reducing visitor anxiety lowers the barrier to action (ThrillX methodology)
- **Changes:** Added "Starting from $850/night" pricing, cancellation policy badge, "What's Included" checklist, trust signals near CTAs
- **Status:** Live

### V4 — Soft CTA + Problem/Solution
- **File:** `versions/v4-soft-cta.html`
- **Hypothesis:** If we replace hard-sell buttons with lower-commitment language and frame the hero as problem/solution, engagement will increase because "Explore Properties" beat "Book A Call" by 385% in ThrillX case study
- **Changes:** Replaced "Check Availability" with "Explore Properties", hero panels show problem statement (generic hotel stays) and solution (personalized Bahamas experience), simple 2-field inquiry form
- **Status:** Live

### V5 — Best Combined
- **File:** `versions/v5-best-combined.html`
- **Hypothesis:** Combining the winning elements from V2-V4 will outperform any individual variation
- **Changes:** Best headline from V2, anxiety reducers from V3, soft CTAs from V4, merged into ultimate conversion-optimized page
- **Status:** Live

---

## Methodology
Based on Arsh Sanwarwala's ThrillX CRO framework:
- One focused hypothesis per test
- Above-the-fold optimization drives the biggest swings
- Lower-commitment CTAs outperform hard sells
- Address visitor anxieties directly on the page
- Review mining: use real guest language in copy
